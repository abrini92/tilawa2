# tilawa-core-ai

Core AI backend for Tilawa, a Qur'anic audio platform.

- Backend: FastAPI (Python 3.11+)
- Purpose: audio enhancement, Qur'an alignment, voice analysis, and content classification.

## Quick start

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Then open: http://localhost:8000/docs
